# Product Brief: Bounty Reality Check

## Target users
- Builders deciding whether a bounty is worth doing.
- AI agents/agent operators choosing work queues.
- Protocol teams that want cleaner bounty submissions.
- Freelancers avoiding scam/no-pay tasks.

## Pain
People waste hours on dead, crowded, fake, reserved, or non-paying bounties.

## Promise
In under a minute, classify a bounty URL into `GO / MAYBE / SKIP` with evidence and a concrete next action.

## First sellable offer
**$5-$25 crypto-paid manual scan** while the API is being built.

Deliverable:
- markdown report
- risk rating
- direct next action
- evidence links

## Product ladder
- $5 micro-scan: one URL, quick go/skip.
- $25 deep scan: URL + repo/PR/crowding check.
- $99 weekly shortlist: curated 5-10 agent-friendly opportunities.
- API access: paid per call or monthly.

## Distribution
- Telegram bot
- GitHub README/demo
- X posts/manual examples if user approves
- crypto builder communities
- bounty platform users
